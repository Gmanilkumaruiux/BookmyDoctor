BookMyDoctor is a full-stack doctor appointment booking platform that allows patients to search for doctors, book appointments, and manage their healthcare online.

## Features

- User Registration and Login
- Google Authentication
- Search Doctors
- Book Appointments
- Patient Dashboard
- Doctor Dashboard
- Appointment History
- Notifications
- AI Smart Search
- Responsive Design

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
- Google OAuth

## Installation

1. Clone the repository

```bash
git clone https://github.com/Gmanilkumaruiux/Book_My_Dr.git
```

2. Go to the project folder

```bash
cd Book_My_Dr
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
APP_URL=http://localhost:3000
```

5. Start the application

```bash
npm run dev
```
